Course Syllabus

  • Review of Time and Space Complexity
  • Asymptotic Notations: Big-O, Big-Ω, Big-Θ
  • Recurrence Relations
  • Master Theorem
  • Divide and Conquer Technique
  • Recursion and its analysis (Tower of Hanoi,Ackerman Function)
  • Backtracking (N-Queen,Hamiltonian,n-sum of subset problem)
  • Branch and Bound
  • Amortized Analysis
  • Binary Search Trees (Review)
  • AVL Trees
  • Red-Black Trees
  • Splay Trees
  • Trie (Prefix Tree)
  • Segment Trees
  • Heaps and Priority Queues
  • Fibonacci Heap
  • Graph Representation (Adjacency Matrix & List)
  • Breadth First Search (BFS)
  • Depth First Search (DFS)
  • Topological Sorting
  • Minimum Spanning Trees (Prim’s and Kruskal’s Algorithms)
  • Shortest Path Algorithms Dijkstra Algorithm o Bellman-Ford Algorithm o Floyd-Warshall Algorithm
  • Disjoint Set Union (Union-Find)
  • Strongly Connected Components
  • Network Flow Basics
  • String Matching Algorithms o Knuth-Morris-Pratt (KMP) o Rabin-Karp Algorithm o Boyer-Moore Algorithm
  • Dynamic Programming (Advanced Concepts)
  • Greedy Algorithms
  • Sliding Window Technique
  • Two Pointer Technique
  • Monotonic Stack and Queue
  • LRU Cache
  • LFU Cache
  • Design of Custom Data Structures
  • Introduction to Concurrent Data Structures
